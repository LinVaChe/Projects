# Projects

FOR LAB_3

backend/ - FastAPI бэкенд
Проект использует PostgreSQL. В репозитории есть script_create.sql, чтобы можно было быстро развернуть такую же БД локально. Все БД были созданы и заполнены через pg_admin.

Запуск бэкенда 
Из папки lab_3 через терминал командой uvicorn backend.main:app --reload
API документация: http://127.0.0.1:8000/docs

Запуск фронтенда
Из папки lab_3 через терминал командой python -m http.server
Фронтенд будет доступен по адресу: http://localhost:8000
Главная страница магазина по ссылке: http://localhost:8000/static/pages/index.html

Создание базы данных
Не забудьте создать БД прежде, чем тестировать каталог товаров, регистрацию или отзывы. По умолчанию используется:

хост: localhost
порт: 5432
пользователь: postgres
имя базы данных: laba_3
пароль: 'Algov16lu'

Функционал:
Регистрация 
Авторизация (появляется личный кабинет)
Написание отзывов в личном кабинете (Только для авторизованных пользователей)
Просмотр товаров в каталоге и на отдельных страницах
